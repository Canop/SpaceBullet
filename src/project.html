A free game in progress

I just started making this game.

If everything goes according to the plan it will have thousands of levels and will be the most fun and famous game in the interwebs.

But I'm prepared for a more meh future too. In fact I'm just having fun.

What to expect

  • More missions
  • More fun features and game modifiers
  • A mission editor
  • Multi-player game

Compatibility

I develop on Chrome but I expect the game to be OK on Firefox and Safari too. If you see a problem on those browsers, bring it to my attention. I know there will be glitches on IE, I don't want to spend time on that right now.

In the future I might support touch events but the problem of mobile platforms is the screen's size.
